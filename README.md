College Event Management System 🎉

The College Event Management System is a web-based application that helps manage and organize college events efficiently. Students can explore upcoming events and register, while administrators can add, update, and manage events through a dashboard.

Features 🚀
View upcoming events
Event registration system
Add new events
Display event list
Update and delete events
Simple and responsive interface

Run the server
uvicorn main:app --reload
Open index.html in your browser.
